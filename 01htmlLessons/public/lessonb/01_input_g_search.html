Input Search

Why the input element type search does not display the list of items entered in it The <input type="search" > element doesn't display a list of items by itself. To show a list of suggestions as the user types, you need to use the < datalist > element in combination with the list attribute of the < input > element
To dynamically add new words that the user types into the search box and make them available as suggestions, you will need to use JavaScript to update the <datalist> element.
